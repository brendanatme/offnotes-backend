# Offnotes Backend

The Django app which powers the Offnotes app backend.

# System Requirements

- Python 3.12.7
- SQLite

# Installation

1. Set up local python environment using Python 3.12.7
1. Run `pip install -r requirements.txt`
1. Run `python manage.py migrate`
1. Run `python manage.py runserver`
1. Go to `http://localhost:8000/` in the browser
